Trajectory Data Generatory

This is pretty simple tool intended to generate small data sets for testing purposes.
For each path, pick a color, a start time, an end time, and a time delta. Then hit "New Path" to start drawing on the canvas.
Clear all paths and clear last path have pretty straightforward functions.
Use export paths to export the data to a CSV file. The format is X, Y, Date, TrackId.
The start time, end time, and time delta work as follows:

  • Start time is the timestamp of the first point in the path.
  • End time is the timestamp of the last point in the path.
  • Time delta is the time between each point in the path.

The exported data will be sampled such that (end time - start time)/time delta will be the number of points in the path and these are smoothly picked across the path.
